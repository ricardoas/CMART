Initial Configuration


This page details how to initially configure the downloaded Virtual Machines to use C-MART.

Software Installation

The VMs come with the necessary software to run C-MART however it must be installed by the user. The first thing which should be done after launching a VM is to deploy the following appropriate script:

./cmart/deploy/install/cmart_apache_deploy/cmart_apache_deploy.sh
./cmart/deploy/install/cmart_client_deploy/cmart_client_deploy.sh
./cmart/deploy/install/cmart_image_deploy/cmart_image_deploy.sh
./cmart/deploy/install/cmart_mysql_deploy/cmart_mysql_deploy.sh
./cmart/deploy/install/cmart_solr_deploy/cmart_solr_deploy.sh
./cmart/deploy/install/cmart_tomcat_deploy/cmart_tomcat_deploy.sh
The client and populator VMs are the same VM. The VM should be rebooted after the script is run.

Images

The image directory is specified in /cmart/cmartconfig.txt on each tomcat VM. Make sure that the item images are in this directory or are mounted to the directory. If this needs to be changed remember to restart tomcat after making the change by executing:

service tomcat7 stop
service tomcat7 start
The image for a blank item, /cmart/blank.jpg, should also be put in the image directory. After the data base has been populated check to make sure the items and images have loaded properly by accessing http://IPofTomcatVM:8080/cmart-1/index and browsing the website.
